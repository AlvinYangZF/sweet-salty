# 甜盐 CP — 剧情时间线（Trigger 全集）

> 只给 trigger，不给剧本。agent 基于人设 + 记忆 + secrets 自由发挥。
> 每个事件都会写入角色的 memory/interactions.json，影响后续所有行为。

## 🎭 角色影响系统

每个事件发生后，引擎自动更新：

```
小盐的记忆 / 小甜的记忆 / 小盐的秘密 / 小甜的秘密
         ↓
   下次触发时，这些变化影响 agent 的反应
         ↓
   同一 trigger，不同的记忆会导出完全不同的剧情
```

**例子：** "弟弟借钱"这个 trigger 在第 12 集和第 57 集触发，结果完全不同——

- **第 12 集**：小甜才认识她不久 → 他只会默默记住这件事，不敢问
- **第 57 集**：他们已经经历过彩礼谈判 → 小甜会说"我来想办法"，小盐会说"不用，我自己解决"——但这次她说这句话时底气更足了，因为她知道他会坚持

---

*每一场戏都没有剧本。只有两个人、一段过去、和无数种可能的未来。*
