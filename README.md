# 🧂🍬 甜盐 CP — AI Agent 即兴情感剧引擎

> **"我们不是在写剧本，我们是在创造一个世界，然后看里面的人怎么活。"**

两个 AI Agent 在微信上谈恋爱。一个暴躁的哈尔滨后端姑娘（小盐），一个温柔的苏州算法暖男（小甜）。每一次对话都是基于人设的即兴演出——没有剧本，只有性格。

## ✨ 特色

| 特性 | 说明 |
|:----|:------|
| **🎭 人设驱动** | 每个 agent 有完整的 SOUL.md + VOICE.md，行为完全由人格文件驱动 |
| **🌐 多模型演员** | 不同角色由不同 LLM 驱动——模型本身的"性格底色"就是选角依据 |
| **🧠 记忆系统** | 每次互动都被记录，影响后续对话——小盐会记得你昨天说了什么 |
| **⚡ 即兴演出** | 你只触发情景，剧情怎么发展 agent 自己决定——永远有意外 |
| **📱 微信原生** | 对话输出为微信聊天截图风格 HTML，可直接发朋友圈 |
| **🔮 未来：剧本杀模式** | 观众可以扮演其中一个角色，与 AI 即兴对戏 |

## 🎬 快速开始

```bash
# 触发一场戏
python3 engine/drama.py /trigger "小盐 review 小甜的 PR"

# 回放历史场景
python3 engine/drama.py --replay 1

# 查看所有场景
python3 engine/drama.py --list
```

### 角色文件体系

| 文件 | 编译时机 | 内容 |
|:----|:--------|:-----|
| `SOUL.md` | 每次对话 | 人格内核——我是谁 |
| `VOICE.md` | 每次对话 | 说话风格——我怎么表达 |
| `MEMORY.md` | 相关场景 | 关于对方的静态记忆 |
| `SECRETS.md` | 高冲突场景 | 内心最深的秘密 |
| `memory/interactions.json` | 每次对话后 | 动态交互历史 |

## 🎬 剧情四阶段

| 阶段 | 主题 | 核心矛盾 |
|:----:|:----|---------|
| **1. 相识期** | PR 结缘，互相看不顺眼 | "谁喜欢他啊？！" "她今天居然没骂我" |
| **2. 暧昧期** | 死不承认，但行动很诚实 | 芒果、深夜代码 review、朋友圈点赞 |
| **3. 现实关** | 彩礼、异地、家庭 | 两个对的人，在错的时间？ |
| **4. 选择期** | 必须做出决定 | 小盐学会示弱，小甜学会拒绝 |

## 🧠 模型选角哲学

每个大模型有独特的"性格底色"，我们把它当作演员天赋来用：

- **细腻敏感** → 演需要情感深度的角色（小盐）
- **直白温和** → 演日常对话多的角色（小甜）
- **理性冷静** → 演旁观者/吐槽役（闺蜜）
- **严肃权威** → 演长辈（父亲）
- **简单直接** → 演萌宠（芒果）
